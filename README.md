## Understanding Problem Statement

User management dashboard with multi-authentication with Email/Password and Web3 authentication




### Tech Stack Used

# Frontend :-  React , React UI

# Backend :-     Node JS , Express

# Database : Mongo DB




### Install Library:

# clone the project 

# cd server
# npm install
# cd client
# npm install

### How to  run:

# Backend
1. cd server
2. npm run dev

# frontend
1. cd client
2. npm start
